description: Monitor the AI Hive in Real-Time
---

1. Ensure the Hive Simulation is running (or about to run).
   ```bash
   python main.py
   ```

2. Open a new terminal and launch the Dashboard.
   ```bash
   python dashboard.py
   ```

3. Open your browser to the URL displayed (usually http://localhost:5000) to view the "Matrix".
