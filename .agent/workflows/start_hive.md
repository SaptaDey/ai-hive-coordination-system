---
description: Start the AI Hive Coordination System
---

1. Install dependencies (if not already installed).
   ```bash
   pip install flask
   ```

2. Run the main simulation script.
   ```bash
   python main.py
   ```

3. (Optional) To monitor the system, open a new terminal and run the Monitor Hive workflow or:
   ```bash
   python dashboard.py
   ```

4. **Agent Action**: Open the internal browser to `http://localhost:5000` to monitor the Hive Matrix in real-time.
